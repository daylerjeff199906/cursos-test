# courses-test

Lightweight repository for developing and testing course content and examples.

## Contents
- Course materials (lectures, slides, exercises)
- Example code and tests
- Build and preview scripts

## Prerequisites
- Node.js 14+ (or specify project runtime)
- Git

## Quick start
1. Clone the repo:
    git clone <repo-url> .
2. Install dependencies:
    npm install
3. Run local preview:
    npm start

## Project layout
- /docs — markdown course pages and notes
- /examples — runnable code samples
- /exercises — practice problems and starter code
- /tests — automated tests
- readme.md — this file

## Development
- Follow consistent formatting and linting rules.
- Add examples with clear instructions and expected output.
- Keep exercises self-contained and include solutions in a hidden folder or separate branch.

## Testing
- Run unit tests:
  npm test
- Add CI checks (recommended): lint, test, build
